## Build Setup

vue-cliを利用しています。

### ビルド

``` bash
# cloning
git clone https://github.com/s-yoshiki/Gasyori100knockJS.git

cd Gasyori100knockJS

# 必要なモジュール類をまとめてインストール
npm install

# 開発サーバ起動
npm run server

# 本番用ビルド
npm run build

```
