# 学習ガイド

この実装は、完成画像を見るだけでなく「前の問題で作った処理が次の問題でどう使われるか」を
追えるように構成している。初めて画像処理を学ぶ場合は、次のまとまりごとに進めると理解しやすい。

## 1. 画素と近傍処理（Q.1〜24）

最初に `ImageData.data` の RGBA 配列を直接読み書きする。色変換と二値化で一画素の操作を、
平滑化とエッジ抽出で周辺画素を参照する畳み込みを学ぶ。ヒストグラムは画像全体の統計量を扱う入口になる。

確認するとよい点:

- 境界画素をどのように扱っているか
- カーネル係数の合計が明るさへどう影響するか
- 二値化の閾値を固定する場合と、大津法で決める場合の違い

## 2. 幾何変換と周波数（Q.25〜40）

拡大・回転では、出力から入力をたどる逆写像と補間方法の違いを比較する。
フーリエ変換と DCT では、画像を「位置ごとの色」から「周波数の組み合わせ」として見る。

Q.36〜40 は JPEG の各段階を分けているため、DCT、量子化、YCbCr が画質とデータ量へ与える影響を順に確認できる。

## 3. 輪郭・領域・形状（Q.41〜65）

Canny は平滑化、勾配、非極大値抑制、ヒステリシスを連結した処理で、Hough変換は抽出した点から直線を探す。
モルフォロジー、ラベリング、細線化では、二値画像の近傍関係から領域と形状を扱う。

## 4. 特徴量（Q.66〜83）

HOG は勾配をセル単位のヒストグラムへ変換し、正規化して照明変化を抑える。
画像ピラミッド、ガボールフィルタ、Harris/Hessian は、スケール・方向・局所構造という異なる観点で特徴を表す。

## 5. 認識から検出へ（Q.84〜100）

色ヒストグラムによる最近傍分類から始め、k-means、ニューラルネットワークへ進む。
最後は同じ部品を次の順でつなぎ、画像の中から対象領域を探す。

```text
ランダムクロップ → HOG → ニューラルネット → スライディングウィンドウ
                  → 検出候補 → NMS → Recall / Precision / F-score / mAP
```

Q.93 の IoU は、Q.94 の教師ラベル作成、Q.99 の重複除去、Q.100 の評価という
異なる3つの目的で再登場する。数式だけでなく、値がどの判断に使われるかを見るのがポイント。

## ブラウザでの確認方法

1. 問題ページを開き、説明と入力画像を確認する。
2. 必要なら入力画像セレクタを変更する。
3. 「実行」を押し、入力と出力を比較する。
4. 数値を扱う問題では canvas 下のログも確認する。

重い問題を実行中はブラウザのメインスレッドを使用する。Q.32〜35 と Q.97〜100 は、
ほかの問題より完了まで時間がかかることがある。
